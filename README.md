Token Factory

A decentralized application for creating custom ERC-20 tokens on the Ethereum blockchain.

Live Demo
https://mekdelawittkassahun.github.io/TokenFactory/

Overview
Token Factory allows users to deploy their own ERC-20 tokens without writing any code. The application consists of a Solidity smart contract deployed on the Sepolia testnet and a web interface that connects via MetaMask.

Features

Create custom ERC-20 tokens with name, symbol, and supply
Burn tokens to reduce circulating supply
View all tokens created on the platform
Add tokens to MetaMask with one click
Copy token and creator addresses

Technology Stack

Smart Contract: Solidity 0.8.31
Frontend: HTML5, CSS3, JavaScript
Blockchain Interaction: Web3.js
Wallet: MetaMask
Network: Sepolia Testnet

Smart Contract
Contract Address: 0x1640E0C8D3436B7e02FFEedfF0c70554F8d0955B
Network: Sepolia Testnet

Getting Started

Prerequisites
MetaMask browser extension
Sepolia test ETH from a faucet

Steps
Install MetaMask and switch to Sepolia network
Get test ETH from Sepolia Faucet
Visit the Token Factory dApp
Connect your wallet
Create your first token

Usage

Connect Wallet: Click the Connect MetaMask button and approve
Create Token: Enter name, symbol, and supply, then confirm
Burn Tokens: Enter amount and click Burn (creator only)
Add to MetaMask: Click Add to MetaMask on any token card
